Software design, final project

The goal for this porject was to create an application that allow users to define and recognize gestures. To do this, we first created an application that allowed us to record the movements of a person as a collection of 3D poins, each of them representing a body joint in the space. Then, we were able to select the body joints and define rules about them, e.g., "hand 20cm over shoulder", or "knee 1cm over hip" These rules allowed use recognize static postures. Finally we defined a gesture as a sequence of static postures.
